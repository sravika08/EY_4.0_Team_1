How to Create a Schedule
  1. Select Date: Choose the date when the class will be held. You can schedule classes in advance.
  2. Enter Subject: Provide the subject name (e.g., Mathematics, Physics, Chemistry).
  3. Add Topic: specify the topic to be covered (e.g., Chapter 5: Derivatives).
  4. Submit: Click "Create Schedule" to save the class schedule. Once created, you can mark attendance for the scheduled class.
Tips
  • Schedule classes well in advance to help students plan
  • Be specific with topic names for better record keeping
  • You can edit or cancel schedules if needed
  • After creating a schedule, you can mark attendance immediately or later
